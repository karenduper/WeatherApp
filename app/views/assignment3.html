4. Read and explain the following article by Martin Fowler: http://martinfowler.com/articles/richardsonMaturityModel.html. 20 points

It portrays the REST model and its principal elements into three steps seen as levels; those being resources, http verbs and hypermedia controls. And how to make use of them to build integrated systems in an easier way for enterprises using web-style system properties. There’s a level 0 which represents a starting point for this model which uses HTTP as a transport system but without the concepts belonging to the web. Level 1 basically makes requests to individual resources. In Level 2 it uses HTTP verbs the way you would use them in HTTP, no longer like a tunneling mechanism. And in Level 3 it introduces HATEOAS.

5. Explain why cloud computing is important. 10 points.

Because it reduces costs and complexity allowing developers to do big things without investing as much on certain resources, helping democratize high performance computing. It balances the scale between resourceful companies and those without access to those resources.
